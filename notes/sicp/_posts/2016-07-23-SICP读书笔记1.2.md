## 练习1.11

### 递归

``` scheme

(define (f n)
  (if (< n 3)
      n
      (+ (f (- n 1))
         (* 2 (f (- n 2)))
         (* 3 (f (- n 3))))))
```

### 送代

``` scheme
(define (f n)
  (define (iter a b c cnt)
    (if (= cnt 0)
        a
        (iter (+ a
                 (* 2 b)
                 (* 3 c))
              a b
              (- cnt 1))))
  (if (< n 3)
      n
      (iter 2 1 0 (- n 2))))
```


## 练习1.12

``` scheme
(define (pascal x y)
  (cond ((= x 0) 1)
        ((= y 0) 1)
        ((= x y) 1)
        (else (+ (pascal (- x 1) (- y 1))
                 (pascal (- x 1) y)))))
```
